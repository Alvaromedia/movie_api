Request URL HTTP Method Response
Get a list of all movies /movies GET Returns a JSON object of all movies.
Get data about a specific movie /movies/:title GET Returns data about a specific movie.
Return data about a genre by name/title. /genres/:genre GET Returns data about a genre (e.g.,'Thriller')
Get data about director /directors/:name GET Returns data about the director
Add new user /users/:createUser POST Adds new user to JSON object.
Allow users to update their user information /users/:modifyUser PUT Returns the data modifyed by the user
Add movie to favourites /favourites/:favouriteMovie POST Returns a text confirming which movie was added successfully.
Delete movie from favourites /favourites/:deleteMovie DELETE Returns a text confirming which movie was removed successfully.
Delete user /users/:deleteUser DELETE Returns a text confirming that the user was deleted.
